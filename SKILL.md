# 小学奥数出题器

一句话出一套**例题 + 练习 + 作业 + 参考答案**的奥数练习，自动配**课本风格几何图**，并落成 Word 文档。

## 和"随便让 AI 出题"不同的三个硬保证
1. **图用代码画，不用文生图** —— 调用本 skill 自带的 `scripts/draw_figure.py`（米色方格纸·黑色粗轮廓·45°斜线阴影·双箭头标注），风格稳定一致；几何题**严禁**用 DALL·E/豆包/MJ 等文生图（数字会乱、尺寸会错）。
2. **答案用代码算 + 交叉校验** —— 每道题的数值结果必须用 Python 计算并用两种方法对照，全部通过才写进文档，不靠心算。
3. **图文数据同源** —— 图里的尺寸、题干里的数字、校验脚本里的变量，是同一组，绝不打架。

## 需要用户提供（缺哪个问哪个）
1. 年级（1–6）
2. 章节（如"巧求面积""和差倍问题""植树问题"）
3. 难度（如"举一反三""小蓝本""高斯课本"）
   - 难度锚定：举一反三 ≈ 课内拔高；小蓝本 ≈ 竞赛入门；高斯课本 ≈ 竞赛进阶。
   - 拿不准时，请用户发一页样题对齐难度，不硬猜。

## 第 0 步 · 依赖自检
两个脚本已内置自检，缺库会自动 `pip install matplotlib python-docx`。出图用 matplotlib（纯 Python、无需浏览器），在 OpenClaw 的 host 或 sandbox/容器里都能跑。脚本路径以本 skill 目录为准（`scripts/draw_figure.py`、`scripts/build_doc.py`）。

## 执行流程（SOP）

**A. 接任务**：核对 年级 / 章节 / 难度，缺啥问啥。

**B. 出 6 道例题（默认批量模式）**：一次性给出 6 道文字版例题，覆盖本章核心知识点。每道含：题目 / 分析 / 解。有图的题标【需配图】，并准备好图形数据（顶点 + 标注，单位 cm）。
- 格式铁律：全中文 + Unicode 符号（× ÷ a² ½），**不用 LaTeX**；**不用下标字母**（写 A，不写 B₁）；数据合理（面积/长度为整数结果，不出现"半个人"）。
- 想逐题确认的用户：改为逐题发、逐题确认；要改的记下重出。

**C. 用户确认例题**：用户标"可以 / 要改"，要改的重出。

**D. 配图（有图的题）**：为每道图题写一个 spec.json，调用出图引擎：
```bash
python3 scripts/draw_figure.py spec_例1.json
```
spec 示例：
```json
{
  "polygon": [[0,0],[10,0],[10,3],[6,3],[6,6],[0,6]],
  "shade": true,
  "dims": [
    {"from":[0,0],"to":[10,0],"label":"10cm"},
    {"from":[0,0],"to":[0,6],"label":"6cm"},
    {"from":[6,3],"to":[10,3],"label":"4cm","gap":1.4},
    {"from":[10,0],"to":[10,3],"label":"3cm"}
  ],
  "out": "fig_例1.png"
}
```
引擎自动：把标注甩到图形外留白（延长线 + 双箭头）、按内容算边距防裁切、画课本风格。**先把 PNG 发给用户看，确认后再进文档。图里的数字必须与题干、与答案校验同源。**

**E. 出练习和作业**：每个例题配 2 道同难度练习；作业共 12 道，统一编号"作业一 ~ 作业十二"，不分组、不标"例X作业"。

**F. 代码校验答案**：把每道题（例题/练习/作业）的答案用 Python 算一遍并**交叉验证**（如面积用"大减小"和"分割相加"两法对照、用鞋带公式复核），全部一致才写入。把校验过程留痕。

**G. 生成 Word**：把内容整理成 problems.json，调用：
```bash
python3 scripts/build_doc.py problems.json
```
结构：例1（题/分析/解/练习）… 例6 → 作业一~十二 → 参考答案（练习 + 作业全部答案）。有图例题图片嵌在题目下方。文件名：`{年级}年级_{章节}_{难度}_练习题.docx`。

**H. 交付**：飞书 / 微信发送文件；拿不到发送渠道就直接给出本地文件路径。

## 出图引擎（两个,按题型选）
**几何题严禁文生图。AI 的职责是"算出坐标 + 写 spec.json",渲染交给脚本。坐标必须与题干、与答案计算同源。**

### A. scripts/draw_figure.py —— 割补 / 面积 / 方格类
- 输入 JSON：`polygon`（顶点,cm）/ `shade` / `dims`（每条 from,to,label，可选 `gap`、`dir`）/ `out`。
- 适合：割补求面积、组合图形、阶梯 / L 形。课本风格（米色方格纸·45°阴影·延长线双箭头·自动防裁切）。

### B. scripts/geo_figure.py —— 初中几何 / 坐标系 / 函数题
- 输入 JSON：
  - `points`：命名点 `{"A":[0,4],"B":[0,0],...}`（坐标由 AI 计算,如垂足、中点）
  - `polygon`：用点名连成多边形（可 `shade`）
  - `segments`：连线 `[["A","E"],["B","F"]]`
  - `right_angles`：直角标记 `[["F","B","A"]]`（在 F 处,BF⊥FA 画小方块）
  - `seg_labels`：边长标注 `[{"seg":["B","E"],"text":"1"}]`
  - `ticks`：等长刻度；`labels`：点名方位 `{"A":"ul","E":"down"}`
  - `axes`：坐标系 `{"x":[-2,4],"y":[-2,5]}`（自动画箭头/O/刻度/网格）
  - `funcs`：函数图象 `[{"k":-1,"b":3,"label":"y=..."}]`(一次) 或 `{"a":1,"b":0,"c":-2}`(二次) 或 `{"points":[...]}`
  - `circles`：圆 `[{"c":"O","r":2}]`（`c` 可填点名或坐标,`dash:true` 画虚线）
  - `arcs`：弧 `[{"c":"O","r":2,"a0":0,"a1":120}]`；`sectors`：扇形 `[{"c":"O","r":2,"a0":0,"a1":90,"shade":true}]`
  - `angles`：角标记 `[{"v":"B","p1":"A","p2":"C","r":0.5,"text":"30°"}]`
- 适合：三角形、四边形 + 构造线、一次/二次函数图象、圆与切线、扇形、几何证明配图。
- 切线题画法：画 `circles` + 切点和外点放进 `points` + `segments` 连切线 + `right_angles` 在切点标半径⊥切线。
- ⚠️ 构造题(垂足/交点/中点/切点)先用 Python 算出坐标,再填进 `points`——画图的坐标 = 解题的坐标。

## 边界与注意
- 几何题**不用文生图**。
- 答案必须**代码校验**后才输出。
- 难度拿不准先用样题对齐，不硬猜。
- 题目是初稿，提醒老师按本班学情二次把关，不夸大、不承诺提分。
